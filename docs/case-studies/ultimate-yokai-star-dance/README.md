# Ultimate Yokai Star Dance Case Study

This folder stores persistent notes and historical change-log snapshots for the
Ultimate Yokai Star Dance progression workflow.

## Current Scope

- Encounter: Ultimate Yokai Star Dance
- Phase: P1
- Active baseline: `v0.1-draft`
- Current status: short-progression draft workflow
- Confidence: draft until observed mechanics and targeting rules are confirmed

All generated guides in this folder are progression drafts. They must be corrected against live pulls, logs, screenshots, or verified team notes before public sharing.

## Repository Layout

- `xivplan-ffxiv-guide/assets/progression-notes/ultimate-yokai-star-dance/raw-notes/`:
  source notes from user input, screenshots, timeline snippets, and calls.
- `artifacts/ultimate-yokai-star-dance/`: local generated outputs, including
  parsed IR, knowledge matches, solution candidates, generated specs, `.xivplan`
  exports, guide packages, quality reports, and the runtime change log.
- `docs/case-studies/ultimate-yokai-star-dance/change-log.md`: tracked
  historical snapshot of version-by-version deltas.

## Version Policy

- `v0.1-draft`: based on oral notes, first-look information, or placeholders.
- `v0.2-observed`: adds observed targeting rules and resolution order.
- `v0.3-stabilized`: standing positions and movement routes are stable.
- `v1.0-release`: public guide package.

New information should update the source notes first, then generate a new
version. Older version outputs are retained locally under `artifacts/`.
