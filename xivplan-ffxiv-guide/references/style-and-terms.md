# Style And Terms

## Language

Use Simplified Chinese. Write for CN FFXIV/FF14 players. Keep wording practical and direct:

- Prefer "怎么处理" over lore or background.
- Prefer short paragraphs and tables for role responsibilities.
- Explain uncommon shorthand once, then use the shorthand.
- Do not bury movement timing inside long prose.

## Writing Rules

- Start every full mechanism with a one-sentence summary.
- Use "图 1/图 2/图 3" only when there are multiple figures; otherwise name the single figure directly.
- For towers, specify who soaks each tower and whether towers are one-person or multi-person.
- For stacks, specify stack size and who joins.
- For spreads, specify fixed clock positions or priority rules.
- For knockbacks, specify source, direction, landing zone, and whether anti-knockback is allowed.
- For tethers, specify both endpoints, pull direction, crossing restrictions, and when to return.
