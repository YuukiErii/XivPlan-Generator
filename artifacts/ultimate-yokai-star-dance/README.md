# Ultimate Yokai Star Dance Short Progression Workspace

This folder stores versioned draft artifacts for Ultimate Yokai Star Dance.

## Current Scope

- Encounter: Ultimate Yokai Star Dance
- Phase: P1
- Active baseline: `v0.1-draft`
- Current status: short-progression draft workflow
- Confidence: draft until observed mechanics and targeting rules are confirmed

All generated guides in this folder are progression drafts. They must be corrected against live pulls, logs, screenshots, or verified team notes before public sharing.

## Folder Layout

- `raw-notes/`: source notes from user input, screenshots, timeline snippets, and calls.
- `parsed-ir/`: mechanic IR and timeline IR generated from notes.
- `knowledge-matches/`: local analogy search reports.
- `solution-candidates/`: candidate strategies, scores, comparison reports, and risk files.
- `generated-specs/`: generated XivPlan scene specs.
- `generated-xivplan/`: `.xivplan` files and exported PNG manifests.
- `guide-packages/`: Markdown, DOCX, PDF, and copied scene/spec files.
- `quality-reports/`: per-version quality gate reports.
- `change-log.md`: version-by-version deltas.

## Version Policy

- `v0.1-draft`: based on oral notes, first-look information, or placeholders.
- `v0.2-observed`: adds observed targeting rules and resolution order.
- `v0.3-stabilized`: standing positions and movement routes are stable.
- `v1.0-release`: public guide package.

New information should update `raw-notes/` first, then generate a new version. Older version outputs are retained.
